# Dash Web Application

Мультистраничное веб-приложение на Dash для обработки и анализа данных.

## Функциональность

- **Кривая доходности** - визуализация и анализ кривых доходности
- **Транскрибация** - преобразование аудио/видео файлов в текст
- **Распознавание** - извлечение текста из изображений и PDF
- **Новости** - управление и отображение новостей по компаниям

## Технологии

- Python 3.12
- Dash Framework
- Plotly для визуализации
- Gunicorn для production сервера
